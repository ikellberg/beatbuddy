## Код-ревью: US-019 — Полноэкранные edge-вспышки по зонам точности

### Вердикт: Готов к коммиту

### Замечания к коду

1. **[ЗАМЕЧАНИЕ]** **`src/web/js/locations/RunnerLocation.js`, строка 69**: Цвета `perfect` и `good` в edge-вспышке субъективно различаются слабо на части экранов.
   - Сейчас: `perfect: 0x43D66E`, `good: 0xFFC94D` + `ADD` blend (строка 219) визуально сближает яркость на светлом фоне.
   - Нужно: усилить разделение (например, сделать `good` более оранжевым и/или немного снизить `peakAlpha` для `good`, либо добавить дополнительный визуальный признак для `perfect`, например короткий второй импульс).
   - Причина: AC требует мгновенно понимать качество попадания по цвету (`perfect = зелёный`, `good = жёлтый`, `miss = красный`), а по факту ручной прогон показал слабую различимость `perfect/good`.
